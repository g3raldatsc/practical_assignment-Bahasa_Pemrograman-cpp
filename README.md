# Product Catalog Management
A C++ console program to manage a product catalog stored in a .csv file.

---

## Description
This program provides simple CRUD (Create, Read, Update, Delete) functionality for managing a product list. All data is read from and saved to the katalog_produk.csv file automatically.

Each product is managed with the following attributes:

- SKU (Stock Keeping Unit)

- Product Name

- Category

- Price

Available menu:

- Display All Products

- Add New Product

- Update Product Data

- Delete Product

- Exit

## How to Run
1. Clone or download this repository.

2. Ensure you have a C++ compiler (e.g., G++ or MinGW).

3. Open your terminal and navigate to the project directory.

4. Compile the program (assuming the file is named katalog.cpp):
``` bash
g++ katalog.cpp -o katalog_app
```
5. Run the program
``` bash
./katalog_app
```
